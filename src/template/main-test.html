CV Screening Dashboard

Chấm điểm CV theo JD, sắp xếp theo mức phù hợp cao nhất

Upload CV trước để lấy id, upload JD tiếp theo, sau đó chạy process để xem danh sách ứng viên được sắp xếp giảm dần theo overall score. Mỗi ứng viên có phần xem chi tiết để so sánh skills, education và experience với JD.

Luồng xử lý
Upload CV → Upload JD → Run → Xem ranking
Dữ liệu hiển thị
Overall score, final reasoning, bảng so sánh chi tiết

Điều khiển

Điền dữ liệu theo đúng thứ tự để tạo id và sinh kết quả so sánh.

1. Upload CV

Field name phải là files để khớp với /upload_cv/.

2. Upload JD

Field name phải là file. JD chỉ upload được sau khi có CV id.

3. Run Process

Nút này gọi /process/{id} và hiển thị ranking giảm dần theo score.

4. Dev mode

Mở trang này để xem toàn bộ nội dung trả về từ API debug.
/devmode
